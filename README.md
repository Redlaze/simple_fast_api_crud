# Simple FastAPI CRUD (Books API)

Простой, но функциональный пример CRUD-приложения на FastAPI для управления каталогом книг. Проект демонстрирует асинхронную работу с базой данных, валидацию данных, middleware для логирования и пагинацию.

## Возможности

*   **Управление книгами**: Создание, чтение и удаление записей о книгах.
*   **Асинхронная SQLite**: Использование SQLAlchemy для асинхронной работы с базой данных.
*   **Валидация Pydantic**: Четкие схемы для входящих и исходящих данных.
*   **Пагинация**: Встроенная поддержка параметров `limit` и `offset` для списка книг.
*   **Middleware для логирования**: Автоматическое логирование всех входящих запросов и времени их обработки.
*   **Утилита для сброса БД**: Специальный эндпоинт для быстрой перестройки базы данных (удобно для разработки и тестирования).
*   **Интерактивная документация**: Автоматически генерируемая документация Swagger UI (по адресу `/docs`).

## Стек технологий

*   **Python 3.13**
*   **FastAPI** — современный веб-фреймворк
*   **SQLAlchemy 2.0+** — асинхронная ORM для работы с БД
*   **SQLite + aiosqlite** — легковесная асинхронная база данных
*   **Pydantic** — валидация данных и управление настройками
*   **Uvicorn** — ASGI-сервер для запуска приложения

## Детали реализации
### Логирование (Middleware)
Все HTTP-запросы проходят через кастомный middleware (src.middleware). В консоль выводятся:

* Входящий запрос с уникальным ID.
* Исходящий ответ с тем же ID, HTTP-статусом и временем обработки в секундах.
* **Обработка ошибок**: если в процессе возникло исключение, оно логируется, а затем пробрасывается дальше.

### База данных
* Используется SQLite с асинхронным драйвером aiosqlite.
* Файл базы данных (my_database.db) создается автоматически в корне проекта при первом обращении к БД.
* Зависимость SessionDep (Annotated[AsyncSession, Depends(get_session)]) используется во всех роутах для получения сессии БД.
